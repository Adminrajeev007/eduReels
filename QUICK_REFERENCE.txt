      2. Click "Create new secret key"
      3. Copy and paste into .env as OPENAI_API_KEY=sk-...
      
   Hugging Face:
      1. Go to: https://huggingface.co/settings/tokens
      2. Click "New token"
      3. Select "read" permission
      4. Copy and paste into .env as HUGGINGFACE_TOKEN=hf_...

🎯 PIPELINE FLOW (9 Steps):

   Input: Degree → 
   1. Cache Check (instant if found)
   2. Research Agent (find topic)
   3. Question Generator (create question)
   4. Quality Checker (validate score ≥7)
   5. Regenerate if needed (max 2x)
   6. Answer Generator (create answer)
   7. Build Payload (combine all)
   8. Cache Result (store for future)
   9. Return → Output: Complete reel package
