┌────────────────────────────────────────────────────────────────────────────┐
│ 🆘 ERROR HANDLING & FALLBACKS                                             │
└────────────────────────────────────────────────────────────────────────────┘

Every agent has 3 layers of resilience:

LAYER 1: Primary Model
  └─ Use ChatGPT (fast, powerful)
  └─ Parse JSON response
  └─ Validate structure

LAYER 2: Fallback Model
  └─ If ChatGPT fails or rate-limited
  └─ Switch to Hugging Face (free)
  └─ Same JSON parsing & validation

LAYER 3: Heuristic Fallback
  └─ If both models fail
  └─ Use curated fallback values
  └─ Research: pre-defined topics
  └─ Generator: generic question template
  └─ Quality: heuristic scoring (Why/How → 8.0, What-is → 4.0)

Result: System NEVER fails, always returns quality content
